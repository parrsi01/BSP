# BeatStars — Complete Producer Playbook

## 1. Account Tiers — Which Plan Is Worth It and When

BeatStars offers three main tiers. The naming and pricing has evolved; use current beatstars.com for live pricing, but the feature split below is accurate.

| Feature | Free | Basic (~$9.99/mo) | Unlimited (~$29.99/mo) |
|---------|------|------------------|----------------------|
| Beats you can upload | 10 | 100 | Unlimited |
| Custom storefront URL | No | Yes | Yes |
| License auto-delivery | Yes | Yes | Yes |
| Commission to BeatStars | Yes (high %) | Lower % | Lowest % |
| Email marketing tool | No | Limited | Full access |
| Discount codes | No | Yes | Yes |
| Analytics dashboard | Basic | Standard | Advanced |
| Custom domain | No | No | Yes |
| Stem file delivery | No | Yes | Yes |
| Collab (split payment) | No | No | Yes |
| Beat of the Week eligible | No | Yes | Yes |
| Affiliate program | No | Yes | Yes |

### Which tier at which stage

| Your situation | Right tier |
|---------------|-----------|
| Just starting, no catalog yet (<10 beats) | Free — validate the platform costs nothing |
| Consistent uploads (1–2 beats/week), any sales | Basic — custom URL and lower fees pay for the plan within 1–2 sales |
| 30+ beats, active email list, making $100+/month | Unlimited — the email tool, advanced analytics, and custom domain justify it |

**The math on commission:** BeatStars takes a percentage of each sale on the free tier (historically around 30%). On paid plans this drops significantly. A single $49.99 WAV sale at the lower commission rate on Basic recoups roughly half the monthly cost. If you sell 2+ leases per month, Basic pays for itself.

## 2. Profile Optimization — Every Field

Your profile is a landing page. Treat it as conversion real estate.

| Field | What to write | What NOT to do |
|-------|--------------|----------------|
| Producer name | Your artist name — consistent with every platform | Don't use your real name unless it is your brand |
| Profile photo | Head shot or logo, 1000x1000px minimum, no pixelation | Random beat art or a smartphone blur |
| Banner | 2560x1440px — brand name, genre tags, short CTA | Leave blank or use a generic template |
| Bio (about) | Genre + DAW + who you've worked with + "Contact for custom beats" — 80–120 words | Life story or copy-pasted text from another platform |
| Genre tags | Select all relevant subgenres: House, Deep House, Tech House, Afro House, Trap, Dancehall | One generic tag only |
| Social links | Link every active platform (Instagram, TikTok, YouTube, SoundCloud) | Dead links or links to inactive profiles |
| Contact email | A dedicated producer email (not personal) | No email listed at all |
| Website URL | Your Carrd or custom site, or your Linktree | Leave blank |

### Bio formula that converts

```
[Genre] producer from [City/Country].
Making [subgenre] beats since [year].
DAW: [FL Studio / Ableton / etc.].
[Optional: notable client / feature / placement].
Available for custom beats and licensing inquiries — DM or email below.
```

Example:
```
Deep house and afro house producer. Based in Cape Town.
Making club-ready beats since 2021. FL Studio.
Custom beats available — DM for rates.
```

## 3. Beat Page Optimization

Each individual beat page is its own mini landing page indexed by BeatStars search.

### Title structure

```
[Subgenre] Type Beat - "[Beat Name]" | [BPM] BPM | [Key]
```

Examples:
- `Deep House Type Beat - "Harbour Lights" | 124 BPM | F Minor`
- `Afro House Type Beat - "Kalahari" | 126 BPM | A Minor`
- `Trap Dancehall Type Beat - "Kingston Nights" | 140 BPM | G Minor`

The phrase "Type Beat" is searched heavily on BeatStars. Include it even though you are BSP originals — it signals to artists the intended genre and use.

### Tags — use every available slot

Fill all tag fields. Priority order:

1. Primary genre (House, Trap, Dancehall)
2. Subgenre (Deep House, Afro House, Trap Dancehall, Riddim Dancehall)
3. Mood (Dark, Groovy, Energetic, Melodic, Euphoric, Hypnotic)
4. Tempo descriptor (120 BPM, 126 BPM, 140 BPM — BeatStars allows BPM as a tag)
5. DAW keywords (FL Studio, Ableton — producers search this to study your setup)
6. Related artist names (not trademarked misuse — framing like "influenced by" is safer than "Afrobeats [Artist] type")
7. Trending keywords from the BeatStars search bar autosuggest (check weekly)

### Description — 80 to 150 words

Structure:
```
Line 1–2: The vibe and genre ("A late-night deep house groove with warm Rhodes chords and a rolling 4/4 kick...")
Line 3: Technical specs (BPM, key, mix notes)
Line 4: What's included in each license tier
Line 5: CTA ("Purchase the lease above or DM for a custom exclusive")
Line 6: Tags block (repeat key terms naturally)
```

### Cover art
- 1000x1000px, JPEG
- Legible at thumbnail size (100x100px on mobile)
- Consistent visual brand across your catalog — buyers recognize your style in the marketplace grid
- No cluttered stock photos — minimal, bold typography works best

### Audio preview length
- Set to 60–90 seconds for individual beats
- The preview should include: 8 bars of intro, the drop/main section, one breakdown
- Beat tags (watermarks) should appear at :30, :60 in the preview — audible but not intrusive

## 4. BeatStars SEO — How Internal Search Works

BeatStars search pulls from: beat title, genre tags, mood tags, BPM, description text, and producer name. It also weights plays, sales, and recency.

### The keyword strategy

**Research method:**
1. Open BeatStars search bar → type "deep house" — note the autocomplete suggestions
2. Those suggestions are real search queries from buyers
3. Build your title and tags around the exact phrases buyers type

**Top-searched terms for house beats (based on platform patterns):**
- deep house type beat
- afro house beat
- tech house beat free
- house music instrumental
- 124 BPM house beat
- melodic house beat
- deep house beat for sale
- soulful house beat

**Top-searched terms for trap dancehall:**
- trap dancehall type beat
- dancehall riddim 2024/2025
- jamaican dancehall beat
- afrobeats dancehall
- trap dancehall instrumental
- dancehall beat for sale
- 140 BPM dancehall

### SEO optimization rules
- Match the most-searched phrase exactly in your title
- Repeat the genre phrase naturally once in the description
- Never keyword-stuff the description — BeatStars penalizes spam-like pages
- Upload consistently — recency is a ranking factor; a new beat gets a temporary boost window

## 5. The BeatStars Marketplace

The BeatStars Marketplace is the browsable beat store where buyers discover producers without searching a specific name.

### How to appear in it
- Your beats are automatically listed in the Marketplace when they are set to "Public" and "For Sale"
- Category placement is determined by your genre tags — accurate tagging is the gating factor
- Beats with more plays, likes, and sales rank higher within each category
- Paid plans get additional marketplace visibility features

### Category placement strategy
- Tag for the most specific category you fit — "Afro House" beats appear in both Afro House and House
- A beat tagged only as "Electronic" will appear in a massive undifferentiated pool — avoid
- Use the mood category (Groovy, Dark, Energetic) to appear in mood-filtered searches

## 6. Pricing Psychology on BeatStars

The 3-tier display is built into BeatStars product pages. Every beat shows all license options side by side. This is anchoring — use it deliberately.

### The 3-tier setup

| Tier | Price point | Psychology |
|------|------------|-----------|
| Basic MP3 | $29.99 | Entry point — makes middle look reasonable |
| Premium WAV | $49.99 — mark as "Most Popular" | The actual target purchase |
| Unlimited | $99.99 | Makes WAV feel like a deal; some buyers do choose it |

### Bundle deals on BeatStars
- Create Collections (available on paid plans) — group 3–5 beats as a themed bundle
- Price the bundle at 25–35% below buying each individually
- Name bundles specifically: "Afro House Vol. 1 — 5-Beat Pack" converts better than "Bundle Deal"

### Flash sale setup
1. BeatStars settings → Pricing → apply a percentage discount to individual beats or the whole catalog
2. Set a start and end date for the sale period
3. The original price shows crossed out next to the sale price automatically — this is the visual anchor
4. Announce on Instagram Stories with the countdown sticker pointing to your BeatStars store
5. Rule: never run a sale longer than 48 hours; urgency drives the purchase decision

## 7. BeatStars Exclusive Features

### Contract generator (auto-license on purchase)
When a buyer completes a purchase, BeatStars automatically generates and delivers a license contract to them. This contract is based on the license template you set in your settings.

**What to customize in your license template:**
- Your legal name or business name
- Credit requirement language: "Must credit 'Produced by BSP' in all releases"
- Stream and download limits per tier
- Governing law (your country/jurisdiction)
- Prohibition on sublicensing or reselling

Review your template whenever you change your pricing tiers.

### Stems delivery automation
- Upload a ZIP file of your stems (individual track bounces: kick, bass, melody, etc.) to the beat page
- Set which license tiers receive the stems on purchase
- BeatStars delivers the ZIP automatically after payment — you do nothing manually
- Always include stems with Unlimited and Exclusive tier purchases

### Email marketing tool (built-in, paid plans)
BeatStars has an email capture and send feature. Use it to:
- Send "New beat" announcements to your BeatStars follower list
- Run flash sale announcements directly to followers
- Send "Beat of the month" curated picks

This is different from your external email list (Mailchimp/Brevo) — it reaches people who already follow you on BeatStars, which is a warm audience.

### Analytics dashboard — what to track

| Metric | What it tells you | Action threshold |
|--------|------------------|-----------------|
| Profile views | How much traffic your profile gets | Rising = your external marketing is working |
| Beat plays | Which beats get attention | Low-play beats need better titles/tags |
| Sales by beat | Which beats convert | Double down — make similar beats |
| Traffic source | Where visitors come from | Focus effort on the highest-traffic source |
| License type sold | Which tier buyers prefer | Adjust pricing if one tier dominates |
| Conversion rate | Plays → purchases | Below 1% = price or quality issue |

Check analytics weekly. Monthly trend is the meaningful signal — weekly spikes can be noise.

### Discount codes
- Create codes in BeatStars settings → Discounts
- Use cases: DM offer to a specific artist ("Use code NEWTRACK for 20% off"), email list exclusive, collab partner codes
- Track which codes get used — tells you which outreach channel actually converts

### Beat of the Week feature
BeatStars surfaces featured beats in a "Beat of the Week" section. Eligibility on paid plans.
- Not manual — submit your best new beat for consideration
- Focus: high-quality audio, strong artwork, competitive price point
- Even if not selected, submitting keeps your catalog visible to BeatStars curation staff

## 8. Free Methods to Drive Traffic to BeatStars

### SoundCloud → BeatStars
- Every SoundCloud track description: include your BeatStars URL in the first 3 lines
- In the "Buy link" field on every SoundCloud track: paste your BeatStars profile URL or the specific beat page
- Free downloads on SC: replace the free download with a redirect to BeatStars to browse the catalog

### Instagram bio → BeatStars
- Bio link: use Linktree with BeatStars as the first button
- Every Reel caption: "Link in bio → full catalog on BeatStars"
- Instagram Stories: use the link sticker (available to all accounts now) pointing directly to a specific beat on BeatStars

### YouTube description links
- First 150 characters of every YouTube description: your BeatStars URL
- Example: "Buy/lease this beat: [BeatStars beat URL] — Full catalog: [BeatStars profile URL]"
- Add a BeatStars card during the video (YouTube Cards feature at the 30-second mark)

### TikTok link in bio
- TikTok allows one link in bio — use Linktree or a direct BeatStars link
- In every video: "Buy this beat — link in bio" as end-card text
- Comment your BeatStars link on your own videos (pinned comment) since TikTok captions don't have clickable links

## 9. BeatStars Affiliate Program

BeatStars has a referral/affiliate program where you earn a commission for referring new paying producers to the platform.

- Share your referral link (found in your BeatStars account settings)
- When another producer signs up for a paid plan via your link, you earn a commission
- Share the link in: YouTube tutorials, TikTok "how I sell beats" content, blog posts, Discord communities
- This is passive income from your content — a tutorial on "how to set up BeatStars" can drive referrals for months

## 10. BeatStars Chart Ranking

BeatStars has genre charts that show top beats. Ranking higher drives organic discovery.

**What influences chart position:**
- Number of plays (weighted)
- Number of likes
- Number of purchases
- Recency of uploads
- Profile follower count

**Tactics to climb:**
1. Release a new beat on a consistent day/time (Tuesday or Thursday) — builds a play spike
2. Announce each release on all platforms simultaneously for a coordinated play rush
3. Encourage followers to like beats on BeatStars specifically, not just play them
4. Upload regularly — the algorithm rewards active catalogs over dormant ones

## 11. Collab Selling on BeatStars (Co-Producer Splits)

If two producers from BSP make a beat together, BeatStars supports split payments on the Unlimited plan.

- Set the co-producer's BeatStars profile when uploading the beat
- Define the split percentage (e.g., 60/40, 50/50)
- BeatStars distributes the revenue automatically at the split ratio when the beat sells
- Both producers get credit on the beat page

Use this for: beats where one producer made the melody loop and another handled drums and arrangement.

## 12. BeatStars vs Airbit — Decision Table

| Factor | BeatStars wins | Airbit wins |
|--------|---------------|------------|
| Buyer pool size | Yes — much larger | — |
| Free tier feature set | — | Yes — Airbit free has more features |
| Embeddable store widget | Both good | Airbit's widget is slightly cleaner |
| Email marketing built-in | BeatStars (paid) | — |
| Stems delivery automation | Both | — |
| UI simplicity | — | Airbit is cleaner |
| Analytics depth | BeatStars | — |
| Split payments | BeatStars (Unlimited) | — |
| Commission rate | Similar | — |
| Community / social features | BeatStars | — |

**Decision rule:**
- Use BeatStars as your primary store — the buyer audience is larger and worth paying for.
- Use Airbit as a secondary store or embed it on your own website for a clean checkout widget.
- Never maintain two separate catalogs — upload everything to both, price identically, let BeatStars drive sales.
