## Who We Are

BSP is a collective of 4 house music producers focused on:

- Creating high-quality house beats (deep, tech, afro, soulful)
- Selling beats professionally via BeatStars, Airbit, and direct channels
- Building a brand with consistent visual identity
- Supporting each other with shared systems and knowledge

| Role | Responsibility |
|------|---------------|
| Beat Maker A | Primary production, FL Studio 20 |
| Beat Maker B | Primary production, Ableton Live |
| Beat Maker C | Production + mixing, Logic Pro X |
| Marketing Lead / Designer | Graphics, social media, licensing |

## Quick Start

```bash
git clone https://github.com/your-org/bsp.git
cd bsp
```

Then explore by role:

- **Producer:** Start in `tutorials/` for your DAW, then `production/`
- **New to selling:** Start in `marketing/BEAT_SELLING_PLATFORMS.md`
- **Setting up visuals:** Start in `graphics/`

## Contributing

See [.github/CONTRIBUTING.md](.github/CONTRIBUTING.md) for full contribution rules.

**Quick rules:**
1. Never commit binary files (audio, video, large images) — use Google Drive and link.
2. File naming: `BSP_BeatName_BPM_Key_v1.wav`
3. Branch naming: `feature/topic-name` or `fix/topic-name`
4. Commit message format: `type: short description` (e.g. `docs: add afro house tutorial`)
5. One PR per topic. Keep PRs small and focused.
