# House Music Arrangement Structure Guide

---

## Classic House Structure: 8-Bar Phrases

House music is built on **8-bar (8-measure) phrases**. Almost every structural decision — when elements enter, when the drop hits, when the breakdown occurs — happens on an 8-bar boundary.

**Why 8 bars?** DJs mix in 8-bar phrases. Listeners anticipate changes on 8-bar boundaries. Breaking this rule creates tension (intentional or unintentional). Honoring it creates a professional, DJ-playable track.

### The Additive/Subtractive Structure Principle
- **Builds:** Elements are added one or two per 8-bar phrase (additive arrangement)
- **Breakdowns:** Elements are removed one or two per 8-bar phrase (subtractive)
- This creates natural tension/release arcs

## Full Arrangement Map

### Reference: 5-Minute House Track (typical streaming version)

| Section | Bars | Approx. Time | Elements Present |
|---------|------|-------------|-----------------|
| Intro A | 1–16 | 0:00–0:30 | Kick, hi-hats |
| Intro B | 17–32 | 0:30–1:00 | + Bass, minimal percussion |
| Build 1 | 33–48 | 1:00–1:30 | + Chords/pads, riser starts |
| Drop 1 | 49–80 | 1:30–2:30 | Full arrangement, bass pumping |
| Breakdown | 81–96 | 2:30–3:00 | Chords only, no drums |
| Build 2 | 97–112 | 3:00–3:30 | Drums return, filter sweep |
| Drop 2 | 113–144 | 3:30–4:30 | Full arrangement (variation) |
| Outro | 145–160 | 4:30–5:00 | Strip back to drums only |

**For DJ use:** Add 16 bars at the beginning and end of intro/outro (32 bars total intro/outro) so DJs can mix in and out cleanly.

## Intro: Streaming vs DJ Use

### Streaming Intro
- **Length:** 16–24 bars (30–45 seconds at 126 BPM)
- Goal: hook the listener within 30 seconds or they skip
- Can start with the hook or a recognizable element immediately
- Build slightly faster than DJ version

### DJ-Use Intro
- **Length:** 32 bars (approximately 60 seconds)
- First 16 bars: drums only (kick + hi-hats, maybe percussion)
- Second 16 bars: add bass or a single pad — still minimal
- DJ needs at least 16 bars of clean drums to mix the previous track out
- **If you're selling to DJs** — always produce DJ-length intros/outros

### Intro Element Entry Order (Common)
```
Bar 1:  Kick + hi-hats
Bar 9:  + Percussion (shaker, conga)
Bar 17: + Bass
Bar 25: + Chord pad (very low in the mix)
Bar 33: + Main pad / stab at full volume → Drop
```

## Build-Up Techniques

### The Filter Sweep
Most powerful and classic house build technique.

**Method:**
1. Place a low-pass filter on the master or a bus (or the main synth)
2. At the start of the build (8 bars before the drop), set cutoff to 300 Hz
3. Automate the cutoff to rise to 16 kHz by the last beat before the drop
4. Use a slight boost in resonance (10–15%) as the filter opens

**Effect:** The track sounds "muffled" and builds anticipation as frequency content returns.

### Cymbal Roll / Hi-Hat Acceleration
- Layer increasing hi-hat density over the last 4 bars:
  - Bars 1–2 of build: 1/8 note hi-hats
  - Bars 3–4: 1/16 note hi-hats
  - Last 2 beats: 1/32 note hits
- Add a rising cymbal crash with reverb that stretches into the drop

### Noise Riser
- White noise with automated HPF sweep (200 Hz → 16 kHz) over 8 bars
- Volume automation: start silent, arrive at full at drop
- Create anticipation through the rapid increase in high-frequency energy

### The "Strip Back" Build
Counter-intuitive but effective:
1. Remove the kick drum entirely for the last 4–8 bars before the drop
2. Add silence or a single sustained note just before the drop
3. The absence of the kick creates more tension than adding elements
4. **The drop hits harder after silence**

### Tension Chord (Suspended)
- Replace the final chord before the drop with a **suspended chord** (sus4 or sus2)
- Suspended chords feel unresolved — the listener's ear "wants" resolution
- Resolution arrives with the drop → very effective emotional technique

## Drop Energy: Full Arrangement Hit

### The Drop Checklist
Everything should hit together at the drop:

- [ ] Kick returns (if stripped during build)
- [ ] Bass at full level with sidechain pump
- [ ] All drums: kick, clap, hi-hats, percussion
- [ ] Main chord pad/stab
- [ ] Lead melody or hook
- [ ] Any vocal element
- [ ] Sidechain compression fully engaged
- [ ] Filter fully open (if used in build)
- [ ] Reverb tails cleared before the drop

### First Beat of the Drop — Common Mistakes
- **Don't add too much at once:** if the intro + build added elements one at a time, the drop should bring back everything simultaneously but nothing new
- **Avoid flam/timing issues:** make sure the kick and bass are perfectly aligned on beat 1
- **Check mono compatibility:** the drop, with everything playing, is where phase issues most often appear

### Drop Variations (between Drop 1 and Drop 2)
Drop 2 should feel familiar but have at least one change:

| Variation Type | Example |
|---------------|---------|
| New lead melody | Different 8-bar melodic phrase |
| New percussion element | Add cowbell, extra hi-hat layer |
| Chord extension | Replace Am with Am9 or Amaj7 |
| Vocal addition | Vocal hook enters in Drop 2 |
| Filter variation | Drop 2 has more open, brighter filter |
| Arrangement density | Drop 2 has more percussion layers |

## Breakdown: Emotional Drop

### Purpose
The breakdown is an emotional and dynamic contrast to the high-energy drop. It allows the listener (and the dancefloor) to breathe before the second drop hits.

### Classic Breakdown Structure
```
Bar 1–4:   Drums removed; chords play sustained
Bar 5–8:   Melody plays over chords; vocal hook or chop
Bar 9–12:  Minimal — just pads; increasing reverb
Bar 13–16: Near silence or single sustained chord → tension
```

### Breakdown Sound Design Tips
- **Reverb:** Increase reverb wet amount during breakdown (50–80% wet) — creates a "floating" sensation
- **High-pass filter:** Can leave the filter slightly open from before the breakdown
- **Piano or simple instrument:** Adding an unprocessed (dry) piano to the breakdown increases emotional impact
- **Silence before the build:** Leave a full bar of silence or near-silence just before the build begins

---

## Build 2: Re-Tensioning

The second build can be shorter and more aggressive than the first:
- **Length:** 8–16 bars (half the length of Build 1 works well)
- Re-introduce drums quickly (drums appear by bar 4)
- Filter sweep re-engages
- Add a **new element** (vocal sample, extra percussion) to differentiate from Build 1
- End with the same or similar tension moment before Drop 2

## Outro: DJ-Friendly vs Cold Ending

### DJ-Friendly Outro
Required if you want DJs to play your track:

```
Bar 1–16:  Strip elements back to drums + bass only
Bar 17–24: Drums only
Bar 25–32: Kick + hi-hats only
Bar 33+:   Optional: single element fade
```

- **Never fade to silence abruptly in a DJ tool** — the DJ needs clean drums to mix the next track in
- Target: **32 bars of clean drums** at the outro

### Streaming/Album Outro
- Gradual fade over 8–16 bars is acceptable
- Or: a "cold ending" — a final hit, then silence (dramatic effect)
- Cold endings work best for non-DJ releases (SoundCloud drops, album tracks)

### Fade vs Cold Ending

| Context | Ending Type |
|---------|------------|
| DJ release / Beatport | DJ fade (drums to end) |
| BeatStars beat preview | Cold ending or 4-bar loop |
| SoundCloud/streaming | Either — fade preferred |
| Beat tape | Cold ending (more artistic) |
| Sync licensing (TV/film) | Cold ending (to a specific bar) |

## Loop Point Creation for Licensing

Many licensees (content creators, filmmakers) want a "loop point" — a clean section that can loop indefinitely.

### Creating a Loop Section
1. Find the cleanest 4 or 8-bar section in your drop (full groove, no drops or fills)
2. Ensure the last beat connects seamlessly to bar 1 (check the reverb tail doesn't bleed awkwardly)
3. Export this section as a separate file: `BSP_BeatName_BPM_Key_LOOP.wav`
4. Include in the stems delivery for exclusive/unlimited licensees

### Loop Export Checklist
- [ ] Starts on beat 1
- [ ] Ends exactly at bar 4 or 8 boundary
- [ ] No reverb tail that bleeds past the end point (or it's acceptable and won't click when looped)
- [ ] No automation changes within the loop
- [ ] Seamlessly connects when repeated

## DJ-Friendly vs Album-Friendly Comparison

| Element | DJ Version | Album/Streaming Version |
|---------|-----------|------------------------|
| Intro length | 32 bars | 16 bars |
| Outro length | 32 bars | 8–16 bars |
| Total length | 6–8 min | 3–5 min |
| Breakdown | Optional, shorter | More prominent |
| Ending | Drum fade | Cold end or fade |
| Drops | 2 drops | 1–2 drops |
| BPM display | Prominent (DJ needs it) | Less critical |
| Key display | Recommended (harmonic mixing) | Optional |
