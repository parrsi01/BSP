# Daily Checklist — Beat Selling Producers

A practical daily routine for staying productive, consistent, and growing revenue.

---

## Morning Block (15–30 minutes)

### Platform Check
- [ ] Check messages from artists on **BeatStars** (respond to all inquiries same day)
- [ ] Check messages on **Airbit** if active there
- [ ] Check **Instagram DMs** — prioritize artist inquiries over generic messages
- [ ] Check **TikTok notifications** — respond to comments on recent beat videos
- [ ] Check **SoundCloud notifications** — reply to comments and DMs

### Analytics (5 minutes)
- [ ] Open **BeatStars Dashboard** → note total plays, downloads, purchases
- [ ] Check which beat got the most plays overnight
- [ ] Open **Instagram Insights** → note Reel reach and profile visits
- [ ] Note anything significantly up or down for the weekly meeting

### Quick Admin
- [ ] Process any **license deliveries** manually if needed (BeatStars usually auto-delivers)
- [ ] Check if any **exclusive offers** came in — flag for team discussion if high value
- [ ] Note any **artists using your beats** that you should follow up with

---

## Production Block (2–4 hours)

The core creative work. Protect this time — no social media during production.

### Session Start
- [ ] Open your DAW and load the current WIP or start a new project
- [ ] Set the **template** (BPM 124–130, key, time signature)
- [ ] Pull up your **reference track** playlist (see resources/REFERENCE_TRACKS.md)
- [ ] 5-minute ear warm-up: listen to 1 reference track with eyes closed, noting mix details

### During the Session
- [ ] Complete at least 1 beat loop (8 bars of full arrangement)
- [ ] Or: finish the arrangement of a WIP beat (add breakdown, outro)
- [ ] Or: complete the mix of a finished beat

### Beat QC Before Ending Session
If a beat is close to finished:
- [ ] A/B against reference track in same subgenre
- [ ] Check the mix in **mono** (solo the mono output)
- [ ] Listen on **laptop/phone speaker** — check translation
- [ ] Measure **LUFS** with a loudness meter
- [ ] Export a rough version for team feedback if nearly ready
- [ ] Save the project and back it up to the shared drive

---

## Marketing Block (30–60 minutes)

### Content Post (daily — Marketing Lead or rotating)
- [ ] Post **1 beat clip** to Instagram Reels (vertical, 30–60 sec, CTA in caption)
- [ ] Post to **TikTok** (same or different clip, optimized for platform)
- [ ] If it's a new beat release day: post to **YouTube** (visualizer video) and **SoundCloud**

### Engagement (daily — everyone)
- [ ] Like and comment on **10 artist accounts** in your target genre (genuine engagement)
- [ ] Respond to **all new comments** on your posts from the past 24 hours
- [ ] Follow back relevant artist followers

### BeatStars Maintenance
- [ ] Upload new beat if one passed QC (follow upload checklist in team/WORKFLOW.md)
- [ ] Check if any beats are underperforming and need a **description or title update**
- [ ] Refresh the **featured beats** on your profile if applicable

---

## Evening Block (15 minutes)

### Revenue Dashboard
- [ ] Check **BeatStars sales today** (new purchases, licensing revenue)
- [ ] Log revenue in team spreadsheet (or note for weekly meeting)
- [ ] Check if any beats are trending (plays spike = potential for more promotion)

### Tomorrow's Prep
- [ ] Write down **the one beat task** for tomorrow (finish beat X / mix beat Y / start concept Z)
- [ ] Queue **1 social post** for tomorrow morning using Buffer/Later or Instagram scheduler
- [ ] Note any **artist conversations** that need a follow-up DM tomorrow

### Backup
- [ ] Save all DAW project changes
- [ ] If significant work done today: **sync to shared Drive** (Google Drive or Dropbox)
- [ ] New session files: add to the `00_ACTIVE_PROJECTS` folder with proper naming convention

---

## Weekly Variations by Role

### Beat Maker Days (Mon/Wed/Thu — production focus)
Production block extended to 4–6 hours. Marketing block shortened to 20 minutes.

### Marketing Lead Days (Tue/Fri — marketing focus)
- [ ] Batch-create social content for the week (film, edit, schedule all in one session)
- [ ] Update BeatStars descriptions and tags for any new uploads
- [ ] Review analytics and draft the weekly analytics report for the team
- [ ] Reach out to 3 new artists for potential collaboration

### Feedback Day (Friday — all team)
- [ ] Listen to all beats that were sent for review this week
- [ ] Leave written feedback in Discord `#feedback` channel
- [ ] Vote on which beat to feature/promote next week

## Beat Production Session — Deeper Breakdown

For days when the full session is dedicated to a single beat:

| Time | Task |
|------|------|
| 0:00–0:15 | Warm up: listen to 1 reference track |
| 0:15–0:45 | Drum programming (kick, clap, hats, perc) |
| 0:45–1:15 | Bass pattern and chord progression |
| 1:15–1:45 | Lead melody / top line |
| 1:45–2:15 | Arrangement (structure all sections) |
| 2:15–2:45 | Mixing (EQ, compression, sidechain) |
| 2:45–3:00 | Final checks (LUFS, mono, reference A/B) |

If the beat isn't done in 3 hours, **save and continue tomorrow** rather than forcing it. Rushed finishing sounds rushed.

---

## Monthly Checklist (First Monday of the Month)

- [ ] Export **BeatStars revenue report** (CSV) and save to team drive
- [ ] Update the **beat catalog spreadsheet** with all beats uploaded this month
- [ ] Review **social media growth** (Instagram followers, TikTok followers, YouTube subscribers)
- [ ] Identify the **top 3 performing beats** this month — create more in that direction
- [ ] Identify the **bottom 3 performing beats** — consider updating pricing or metadata
- [ ] Review any **artist relationships** that should be followed up with custom beats
- [ ] Check for **expired exclusive licenses** (if time-limited) — can re-list that beat
- [ ] Back up the **full team drive** to a secondary location (external hard drive)
- [ ] PRO registration: register any beats produced this month with ASCAP/BMI
