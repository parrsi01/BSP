## Naming Your Producer Alias or Collective

Your name is permanent — get it right before building a following under it.

### Criteria for a Good Producer Name
- Easy to spell (people search for you)
- Easy to remember (1–2 syllables optimal; 3 max)
- Unique enough to own (check: Google, Instagram, TikTok, BeatStars, YouTube)
- Available as a username across all platforms

### Name Styles That Work for House

| Style | Examples | Notes |
|-------|---------|-------|
| First name + surname | Larry Heard, Kerri Chandler | Classic; feels established |
| Alias (2 words) | Black Coffee, Dixon Avenue | Distinctive, memorable |
| Abbreviation/initials | BSP, MK, DJ EZ | Short, easy to type |
| City reference | Chicago Tracks, Lagos Sound | Genre-anchoring |
| Abstract/evocative | Bicep, Disclosure, Burial | Artistic, ambiguous |

### Checking Name Availability
1. Google "your name" — does anyone else use it?
2. Instagram username search
3. TikTok username search
4. YouTube channel name
5. BeatStars search
6. Twitter/X username

If your exact name is taken, try: `[Name]Beats`, `[Name]BSP`, `[Name]Official`, or `[Name]Music`

## Logo Design

### Free Logo Tools

**Canva** — [canva.com](https://www.canva.com)
- Logo maker: search "music producer logo" in templates
- Fully customizable text + icon combinations
- Export as PNG with transparent background (free)

**Looka** — [looka.com](https://www.looka.com)
- AI logo generator — enter your name and aesthetic preferences
- Generates 100+ variations
- **Free to design; paid to download** (~$65 one-time for logo files)
- Best for getting a polished, brand-ready logo quickly

**Adobe Express** — [express.adobe.com](https://express.adobe.com)
- Free logo maker with Adobe quality
- Good icon library

### Fiverr (Paid) — [fiverr.com](https://www.fiverr.com)
- Budget range: $20–$150 for a decent producer logo
- Search: "music producer logo", "EDM logo", "DJ logo"
- Filter by: **Top Rated Sellers**, reviews 4.8+
- Ask for: vector file (SVG/AI/EPS), PNG with transparent background, all color variations

### Logo File Formats You Need
| Format | Use |
|--------|-----|
| SVG or AI/EPS | Scalable vector — never loses quality; for print and large displays |
| PNG (transparent) | Web, social media overlays, cover art |
| PNG (white on transparent) | Dark backgrounds |
| PNG (black on transparent) | Light backgrounds |
| JPEG | Simple embeds where transparency not needed |

## Color Palette Selection

### Process
1. Go to **Coolors** — [coolors.co](https://coolors.co)
2. Press spacebar to generate random palettes until one resonates with your aesthetic
3. Lock colors you like, keep generating
4. Export your palette as a PNG, HEX codes, or CSS

### Recommended Palette Sizes
- **Primary color:** 1 (your brand color — everything defaults to this)
- **Secondary color:** 1 (accent, used for highlights)
- **Neutral:** 1 (black, white, or dark gray)

**More than 3 colors = inconsistency.** Start with 3.

### Example Brand Palettes for House Subgenres

| Subgenre | Palette Example | HEX |

## Font Pairing

### Finding Fonts
- **Google Fonts** — [fonts.google.com](https://fonts.google.com) — free, web-optimized
- **DaFont** — [dafont.com](https://www.dafont.com) — free fonts including display styles
- **Font Squirrel** — [fontsquirrel.com](https://www.fontsquirrel.com) — free commercial-use fonts
- **Adobe Fonts** — included with Adobe Creative Cloud

### Pairing Rules
1. **Display + Clean:** One expressive/bold headline font + one clean, readable font
2. **Contrast:** If headline is bold, body should be light. If headline is geometric, body can be humanist.
3. **Never:** Two script fonts together. Two very similar fonts together.

### Font Installation
- Download TTF or OTF file
- Double-click to install (Windows/Mac)
- Restart your design application for it to appear

## Consistent Visual Identity Across Platforms

Every platform where BSP appears should look like the same brand:

| Platform | Size | Content | Brand Elements |
|----------|------|---------|---------------|
| Instagram profile photo | 110x110px (display); 320x320px upload | Logo on brand color | Consistent across all platforms |
| Instagram bio | Text | Name, genre, link | Clear CTA |
| TikTok profile photo | 200x200px | Same as Instagram | — |
| YouTube channel art | 2560x1440px | Brand banner + BSP name | Social links |
| YouTube profile photo | 800x800px | Same logo | — |
| BeatStars profile | 1000x1000px | Same or similar to other profiles | Genre description |
| Twitter/X header | 1500x500px | Brand colors + tagline | — |

### Platform Profile Photo — The Rule
Use **the same profile photo** on every platform. This creates instant recognition when an artist sees "BSP" across Instagram, TikTok, and BeatStars — they know it's the same brand.

---

## Profile Photo Standards for All Platforms

### Option 1: Logo Only
- Logo on your primary brand color background
- Simplest, most scalable
- Works at small sizes (16px favicon up to full size)

### Option 2: Photo of Producer (for personal brand)
- Consistent photo across all platforms
- Professional lighting — studio or clean background
- Same photo for at least 6–12 months (recognition builds over time)

### Technical Specs
- Upload at 1000x1000px minimum
- PNG format (better quality than JPEG for logos)
- Platforms will compress — upload the highest resolution you have

## Watermark / Beat Tag Creation

### Audio Tag
Your beat tag is an audio ident that plays during your preview files.

**What to include:** Your name or collective name, said clearly: "B-S-P" or "Beat Selling Producers" or a melodic jingle

**How to record:**
1. Use a microphone or text-to-speech (Eleven Labs, or any TTS engine)
2. Process: light reverb, maybe a delay repeat
3. Export as WAV, keep it under 2 seconds

**Where to place in previews:**
- At 0:30, 1:00, and 1:30 in a 2-minute preview
- Must be prominent enough to identify but not so frequent it's annoying

### Visual Watermark (on videos)
- Semi-transparent PNG of your logo
- Position: lower right corner
- Opacity: 30–40%
- Size: approximately 8–10% of the video width

### On Cover Art
- Logo always present, sized to 5–10% of the image
- Consistent position (always lower right, or always upper left — never both)
